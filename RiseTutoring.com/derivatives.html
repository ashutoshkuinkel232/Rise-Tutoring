DERIVATIVES

Derivatives are a fundamental concept in calculus, representing the rate of change of a function. They have applications in various fields, including physics, engineering, and economics.

What are Derivatives?

 A derivative measures how a function changes as its input changes. It's often described as the "instantaneous rate of change" or the "slope of the tangent line" to a curve at a point.
 Example: If y = f(x), the derivative of y with respect to x is denoted as f'(x) or dy/dx.

Basic Rules of Differentiation:

 1. **Power Rule**: If f(x) = xn, then f'(x) = nxn-1
 Example: If f(x) = x3, then f'(x) = 3x2

 2. **Constant Rule**: If f(x) = c (where c is a constant), then f'(x) = 0
 Example: If f(x) = 5, then f'(x) = 0

 3. **Constant Multiple Rule**: If f(x) = c * g(x), then f'(x) = c * g'(x)
 Example: If f(x) = 3x2, then f'(x) = 3 * 2x = 6x

 4. **Sum Rule**: If f(x) = g(x) + h(x), then f'(x) = g'(x) + h'(x)
 Example: If f(x) = x2 + 3x, then f'(x) = 2x + 3

 5. **Difference Rule**: If f(x) = g(x) - h(x), then f'(x) = g'(x) - h'(x)
 Example: If f(x) = x3 - x, then f'(x) = 3x2 - 1

 6. **Product Rule**: If f(x) = g(x) * h(x), then f'(x) = g(x)h'(x) + h(x)g'(x)
 Example: If f(x) = x * ex, then f'(x) = x * ex + ex * 1 = ex(x + 1)

 7. **Quotient Rule**: If f(x) = g(x) / h(x), then f'(x) = (h(x)g'(x) - g(x)h'(x)) / (h(x))2
 Example: If f(x) = (x2 + 1) / x, then f'(x) = (x * 2x - (x2 + 1) * 1) / x2 = (2x2 - x2 - 1) / x2 = (x2 - 1) / x2

 8. **Chain Rule**: If f(x) = g(h(x)), then f'(x) = g'(h(x)) * h'(x)
 Example: If f(x) = (3x + 2)2, then f'(x) = 2(3x + 2) * 3 = 6(3x + 2)

Higher-Order Derivatives:

 The second derivative, denoted as f''(x) or d²y/dx², measures the rate of change of the rate of change (i.e., the curvature of the function).
 Example: If f(x) = x3, then f'(x) = 3x2 and f''(x) = 6x.

Applications of Derivatives:

 1. **Finding Tangent Lines**:
   • The derivative at a point gives the slope of the tangent line to the curve at that point.
   • Example: Find the tangent line to f(x) = x2 at x = 1. f'(x) = 2x, so the slope is 2. The tangent line is y - 1 = 2(x - 1), or y = 2x - 1.

 2. **Optimisation**:
   • Derivatives are used to find maximum and minimum values of functions.
   • Example: To find the maximum value of f(x) = -x2 + 4x, set f'(x) = -2x + 4 = 0, so x = 2. Then, f(2) = -4 + 8 = 4.

 3. **Motion and Rates of Change**:
   • Derivatives describe the velocity and acceleration of moving objects.
   • Example: If s(t) = t2 + 2t describes the position of an object over time, then s'(t) = 2t + 2 gives the velocity, and s''(t) = 2 gives the acceleration.

Practice Problems:

1. Find the derivative of f(x) = x4.
2. Find the derivative of f(x) = 5x3 + 2x.
3. Find the second derivative of f(x) = x3.
4. Find the derivative of f(x) = (2x + 1)3.
5. Find the derivative of f(x) = ex / x.

Tips and Tricks:

 • Memorize the basic differentiation rules and apply them step-by-step.
 • Practice solving various types of differentiation problems to improve your skills.
 • Use the chain rule when dealing with composite functions.
