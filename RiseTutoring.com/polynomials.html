POLYNOMIALS

Polynomials are expressions consisting of variables and coefficients, combined using addition, subtraction, multiplication, and non-negative integer exponents. Understanding polynomials is fundamental to algebra and higher mathematics.

Definition of a Polynomial:

 A polynomial is an expression of the form:
    anxn + an-1xn-1 + ... + a2x2 + a1x + a0
 where an, an-1, ..., a1, a0 are coefficients and x is the variable.

Types of Polynomials:

 1. **Monomial**: A polynomial with one term (e.g., 5x).
 2. **Binomial**: A polynomial with two terms (e.g., 3x + 2).
 3. **Trinomial**: A polynomial with three terms (e.g., x2 + 4x + 4).
 4. **Polynomial**: A general term for expressions with one or more terms.

Operations on Polynomials

Polynomials can be added, subtracted, multiplied, and divided. Here are the basic operations:

 Addition:
 Combine like terms (terms with the same exponent).
 Example: (3x2 + 2x + 1) + (2x2 + 3x + 4) = 5x2 + 5x + 5

 Subtraction:
 Combine like terms, taking care to subtract each term.
 Example: (5x2 + 3x + 2) - (2x2 + x + 1) = 3x2 + 2x + 1

 Multiplication:
 Distribute each term in the first polynomial to each term in the second polynomial.
 Example: (x + 2)(x + 3) = x2 + 3x + 2x + 6 = x2 + 5x + 6

Factoring Polynomials

Polynomials can often be factored into products of simpler polynomials.

 Common Factor:
 Factor out the greatest common factor.
 Example: Factorise 6x2 + 9x = 3x(2x + 3)

 Difference of Squares:
 This applies to expressions of the form a2 - b2.
 Formula: a2 - b2 = (a - b)(a + b)
 Example: Factorise x2 - 16 = (x - 4)(x + 4)

 Trinomials:
 For quadratic expressions of the form ax2 + bx + c, look for two numbers that multiply to ac and add to b.
 Example: Factorise x2 + 5x + 6:
 Find two numbers that multiply to 6 and add to 5 (2 and 3).
 x2 + 5x + 6 = (x + 2)(x + 3)

 Grouping:
 Group terms to find common factors.
 Example: Factorise x3 + 3x2 + 2x + 6:
 Group terms: (x3 + 3x2) + (2x + 6)
 Factor each group: x2(x + 3) + 2(x + 3)
 Factor out the common binomial: (x + 3)(x2 + 2)

Special Polynomial Formulas

 1. Perfect Square Trinomial:
   • a2 + 2ab + b2 = (a + b)2
   • a2 - 2ab + b2 = (a - b)2

 2. Sum and Difference of Cubes:
   • a3 + b3 = (a + b)(a2 - ab + b2)
   • a3 - b3 = (a - b)(a2 + ab + b2)

Practice Problems

1. Add the polynomials 2x2 + 3x + 1 and x2 - x + 2.
2. Subtract the polynomials 5x2 + 4x + 3 from 7x2 - 2x + 1.
3. Multiply the polynomials (x + 2)(x - 3).
4. Factorise 4x2 - 9.
5. Factorise x2 - 10x + 25.

Tips and Tricks

 • Always simplify polynomials by combining like terms.
 • Recognise special patterns like perfect square trinomials or difference of squares.
 • Practice regularly to become familiar with different polynomial operations.
