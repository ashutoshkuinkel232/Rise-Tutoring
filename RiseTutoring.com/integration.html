INTEGRATION

Integration is a fundamental concept in calculus that represents the accumulation of quantities and the area under curves. It has applications in various fields, such as physics, engineering, and economics.

What is Integration?

 Integration is the process of finding the integral of a function, which represents the area under the curve of a graph of the function.
 Example: The integral of f(x) with respect to x is denoted as ∫f(x)dx.

Basic Rules of Integration:

 1. **Power Rule**: ∫xn dx = (xn+1)/(n+1) + C, where n ≠ -1
 Example: ∫x2 dx = (x3)/3 + C

 2. **Constant Rule**: ∫c dx = cx + C, where c is a constant
 Example: ∫5 dx = 5x + C

 3. **Sum Rule**: ∫(f(x) + g(x)) dx = ∫f(x) dx + ∫g(x) dx
 Example: ∫(x2 + 3x) dx = ∫x2 dx + ∫3x dx = (x3)/3 + (3x2)/2 + C

 4. **Difference Rule**: ∫(f(x) - g(x)) dx = ∫f(x) dx - ∫g(x) dx
 Example: ∫(x3 - x) dx = ∫x3 dx - ∫x dx = (x4)/4 - (x2)/2 + C

 5. **Constant Multiple Rule**: ∫c * f(x) dx = c * ∫f(x) dx
 Example: ∫3x dx = 3 * ∫x dx = 3 * (x2/2) + C = (3x2)/2 + C

 6. **Integration by Substitution**: If u = g(x), then ∫f(g(x))g'(x) dx = ∫f(u) du
 Example: ∫2x * ex2 dx = ∫eu du, where u = x2, du = 2x dx = eu + C = ex2 + C

 7. **Integration by Parts**: ∫u dv = uv - ∫v du
 Example: ∫x ex dx = x ex - ∫ex dx = x ex - ex + C

Definite Integrals:

 A definite integral has limits of integration and gives the net area under a curve between two points.
 Example: ∫ab f(x) dx represents the area under the curve f(x) from x = a to x = b.
 Example: ∫02 x dx = [x2/2]02 = (22/2) - (02/2) = 2

Applications of Integration:

 1. **Finding Areas**:
   • Integration is used to find the area under a curve.
   • Example: Find the area under f(x) = x2 from x = 0 to x = 1: ∫01 x2 dx = [x3/3]01 = 1/3

 2. **Volume of Solids of Revolution**:
   • Integration is used to find the volume of a solid of revolution.
   • Example: Find the volume of the solid formed by rotating y = x2 about the x-axis from x = 0 to x = 1: V = π ∫01 (x2)2 dx = π ∫01 x4 dx = π [x5/5]01 = π/5

 3. **Solving Differential Equations**:
   • Integration is used to solve differential equations.
   • Example: Solve dy/dx = 2x: ∫dy = ∫2x dx, y = x2 + C

Practice Problems:

1. Evaluate ∫x3 dx.
2. Find ∫(2x + 3) dx.
3. Evaluate ∫13 (2x) dx.
4. Use substitution to evaluate ∫x ex2 dx.
5. Find the area under the curve y = x2 from x = 0 to x = 2.

Tips and Tricks:

 • Practice various techniques of integration to become proficient.
 • Double-check your work, especially when applying the limits of integration.
 • Understand the fundamental theorem of calculus to connect differentiation and integration.
