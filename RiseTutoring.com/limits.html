LIMITS

Limits are fundamental to calculus and mathematical analysis and are used to define continuity, derivatives, and integrals. Understanding limits is crucial for grasping the behavior of functions as they approach specific points or infinity.

What is a Limit?

 A limit is the value that a function (or sequence) "approaches" as the input (or index) approaches some value.
 Example: The limit of f(x) as x approaches a is L, written as limx→a f(x) = L.
   • This means that as x gets closer to a, f(x) gets closer to L.

Basic Properties of Limits:

 1. **Limit Laws**:
   • **Sum Rule**: limx→a [f(x) + g(x)] = limx→a f(x) + limx→a g(x)
   • **Difference Rule**: limx→a [f(x) - g(x)] = limx→a f(x) - limx→a g(x)
   • **Product Rule**: limx→a [f(x) * g(x)] = limx→a f(x) * limx→a g(x)
   • **Quotient Rule**: limx→a [f(x) / g(x)] = limx→a f(x) / limx→a g(x), provided limx→a g(x) ≠ 0
   • **Power Rule**: limx→a [f(x)]n = [limx→a f(x)]n

 2. **One-Sided Limits**:
   • **Right-Hand Limit**: limx→a+ f(x) = L
   • **Left-Hand Limit**: limx→a- f(x) = L
   • If both one-sided limits exist and are equal, the two-sided limit exists.

 3. **Infinite Limits**:
   • If f(x) increases without bound as x approaches a, we write limx→a f(x) = ∞.
   • If f(x) decreases without bound as x approaches a, we write limx→a f(x) = -∞.

Techniques for Evaluating Limits:

 1. **Direct Substitution**: Substitute the value of x into the function.
   • Example: Find limx→2 (3x + 5).
   • Solution: Substitute x = 2 → 3(2) + 5 = 11.

 2. **Factoring**: Factor the function and simplify.
   • Example: Find limx→3 [(x² - 9) / (x - 3)].
   • Solution: Factor to get limx→3 [(x - 3)(x + 3) / (x - 3)] = limx→3 (x + 3) = 6.

 3. **Rationalizing**: Multiply by the conjugate if dealing with square roots.
   • Example: Find limx→4 [(√x - 2) / (x - 4)].
   • Solution: Multiply by (√x + 2) / (√x + 2) to get limx→4 [(x - 4) / ((x - 4)(√x + 2))] = limx→4 [1 / (√x + 2)] = 1/4.

 4. **L'Hôpital's Rule**: Use for indeterminate forms (0/0 or ∞/∞).
   • Example: Find limx→0 [sin(x) / x].
   • Solution: Differentiate numerator and denominator to get limx→0 [cos(x) / 1] = 1.

 5. **Squeeze Theorem**: If f(x) ≤ g(x) ≤ h(x) and limx→a f(x) = limx→a h(x) = L, then limx→a g(x) = L.
   • Example: Find limx→0 [x² sin(1/x)].
   • Solution: Since -x² ≤ x² sin(1/x) ≤ x² and both -x² and x² approach 0 as x approaches 0, by the Squeeze Theorem, limx→0 [x² sin(1/x)] = 0.

Limits at Infinity:

 1. **Limits of Rational Functions**: Compare degrees of numerator and denominator.
   • Example: Find limx→∞ [3x² / (2x² + 1)].
   • Solution: Divide numerator and denominator by x²: limx→∞ [3 / (2 + 1/x²)] = 3/2.

 2. **Horizontal Asymptotes**: If limx→∞ f(x) = L or limx→-∞ f(x) = L, then y = L is a horizontal asymptote.
   • Example: For f(x) = (2x) / (x + 1), as x → ∞, f(x) → 2, so y = 2 is a horizontal asymptote.

Applications of Limits:

 1. **Continuity**: A function f(x) is continuous at x = a if limx→a f(x) = f(a).
   • Example: Determine if f(x) = x² is continuous at x = 2.
   • Solution: limx→2 x² = 4 and f(2) = 4, so f(x) is continuous at x = 2.

 2. **Derivatives**: The derivative of f(x) at x = a is the limit of the average rate of change as the interval approaches zero.
   • Example: f'(a) = limh→0 [f(a + h) - f(a)] / h

 3. **Integral Calculus**: Definite integrals are defined as limits of Riemann sums.
   • Example: ∫ab f(x) dx = limn→∞ Σ [f(xi) Δx]

Practice Problems:

1. Find limx→3 (x² - 9) / (x - 3).
2. Evaluate limx→0 sin(x) / x.
3. Determine limx→∞ (5x - 2) / (3x + 4).
4. Use L'Hôpital's Rule to find limx→0 (ex - 1) / x.
5. Apply the Squeeze Theorem to find limx→0 x² cos(1/x).

Tips and Tricks:

 • Always simplify the function if possible before taking the limit.
 • Familiarize yourself with common limit forms and techniques.
 • Practice using different methods to evaluate limits to build proficiency.
