DIFFERENTIAL EQUATIONS

Differential Equations are mathematical equations that relate some function with its derivatives. They play a crucial role in engineering, physics, economics, and many other disciplines where relationships between variables are expressed in terms of their rates of change.

What is a Differential Equation?

 A differential equation is an equation that involves an unknown function and its derivatives.
 Example: dy/dx = 3x
   • This equation relates the derivative of y with respect to x to the variable x itself.

Types of Differential Equations:

 1. **Ordinary Differential Equations (ODEs)**: Involves functions of a single variable and their derivatives.
   • Example: dy/dx + y = x

 2. **Partial Differential Equations (PDEs)**: Involves functions of multiple variables and their partial derivatives.
   • Example: ∂u/∂t = α∂²u/∂x² (Heat equation)

 3. **Linear Differential Equations**: The dependent variable and its derivatives appear to the power 1.
   • Example: d²y/dx² + p(x)dy/dx + q(x)y = r(x)

 4. **Nonlinear Differential Equations**: The dependent variable or its derivatives appear to a power other than 1 or are multiplied together.
   • Example: dy/dx = y² - x

First-Order Differential Equations:

 1. **Separable Equations**: Can be written as f(y)dy = g(x)dx.
   • Example: dy/dx = xy
   • Solution: Separate and integrate: ∫(1/y) dy = ∫x dx → ln|y| = x²/2 + C → y = ±e^(x²/2 + C)

 2. **Linear First-Order Equations**: Can be written as dy/dx + P(x)y = Q(x).
   • Example: dy/dx + 3y = 6x
   • Solution: Using integrating factor μ(x) = e^(∫P(x) dx) → μ(x) = e^(3x)
   • Multiply through by μ(x): e^(3x) dy/dx + 3e^(3x)y = 6xe^(3x)
   • Integrate: y e^(3x) = ∫6xe^(3x) dx
   • Find y: y = e^(-3x)(2xe^(3x) - 2e^(3x)/9) + C

Second-Order Differential Equations:

 1. **Homogeneous Linear Equations**: d²y/dx² + p(x)dy/dx + q(x)y = 0
   • Example: d²y/dx² - 3dy/dx + 2y = 0
   • Solution: Find characteristic equation: r² - 3r + 2 = 0 → (r-1)(r-2) = 0 → r = 1, 2
   • General solution: y = C₁e^(x) + C₂e^(2x)

 2. **Nonhomogeneous Linear Equations**: d²y/dx² + p(x)dy/dx + q(x)y = g(x)
   • Example: d²y/dx² - y = x
   • Solution: Find particular solution y_p and complementary solution y_c
   • General solution: y = y_c + y_p

Special Techniques for Solving Differential Equations:

 1. **Method of Undetermined Coefficients**: Used for nonhomogeneous linear ODEs with constant coefficients.
   • Example: d²y/dx² - 3dy/dx + 2y = e^(x)
   • Solution: y_c = C₁e^(x) + C₂e^(2x), guess y_p = Ae^(x)
   • Substitute and solve for A: y_p = Ae^(x), y_p' = Ae^(x), y_p'' = Ae^(x)
   • Combine: A e^(x) - 3A e^(x) + 2A e^(x) = e^(x) → A = 1
   • General solution: y = y_c + y_p = C₁e^(x) + C₂e^(2x) + e^(x)

 2. **Variation of Parameters**: Used for nonhomogeneous ODEs.
   • Example: y'' + p(x)y' + q(x)y = g(x)
   • Solution: y = y_c + y_p
   • Assume y_p = u₁y₁ + u₂y₂, where y₁ and y₂ are solutions to the corresponding homogeneous equation.
   • Find u₁ and u₂: u₁' y₁ + u₂' y₂ = 0, u₁' y₁' + u₂' y₂' = g(x)
   • Solve for u₁ and u₂: u₁' = -y₂g(x)/(y₁y₂' - y₁'y₂), u₂' = y₁g(x)/(y₁y₂' - y₁'y₂)

 3. **Laplace Transforms**: Convert differential equations into algebraic equations.
   • Example: y'' + y = δ(t - π)
   • Solution: Take Laplace transform: L{y'' + y} = L{δ(t - π)}
   • Y(s)(s² + 1) = e^(-πs)
   • Solve algebraic equation and take inverse Laplace transform to find y(t).

Applications of Differential Equations:

 1. **Modeling Physical Systems**:
   • Used to model motion, growth, decay, and other physical phenomena.
   • Example: Newton's law of cooling: dT/dt = -k(T - Tₐ)

 2. **Electrical Circuits**:
   • Used to analyze RL, RC, and RLC circuits.
   • Example: Voltage in an RL circuit: L(di/dt) + Ri = V(t)

 3. **Population Dynamics**:
   • Used to model population growth and interactions.
   • Example: Logistic growth model: dP/dt = rP(1 - P/K)

Practice Problems:

1. Solve dy/dx = y cos(x).
2. Find the general solution to d²y/dx² + 4y = 0.
3. Solve the initial value problem: y' + 2y = 3, y(0) = 1.
4. Use Laplace transforms to solve y'' - y = e^(t).
5. Solve the boundary value problem: y'' + y = 0, y(0) = 0, y(π) = 1.

Tips and Tricks:

 • Always check if a differential equation can be simplified or solved using basic techniques.
 • Familiarize yourself with common forms and solutions of differential equations.
 • Practice using different methods to solve various types of differential equations.
